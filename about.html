About GameHub

GameHub is a website project created for Integrative Programming and Technologies (IPT).

It showcases popular games while demonstrating HTML concepts discussed in class.

Learn HTML through gaming content.

This website is intended for educational purposes only.

Project Information

GameHub was developed to demonstrate website organization, multimedia integration, navigation, tables, lists, and other HTML elements.
